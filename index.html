Complete Projects

EPL Acoustic System This is the assembly that houses the 2 speakers, probe tubes, and probe tube microphone used in our EPL Cochlear Function Testing System.
EPL Accelerometer Amplifier This is an amplifier designed to be used with cheap MEMS accelerometers that run off 9V. For use with VSEPs and etc.
EPL Microphone Amplifier This is a part of our larger EPL Cochlear Function Testing System. Two versions exist on for the FG series microphone in that system and another for the EK series used in earlier versions of it, and preferred for it's lower noise in the lower frequency range.
EPL Fiberoptic Amplifier This is a fiberoptic LED with a very stable output and a fiberoptic photodiode with high frequency (~30KHz bw) linear transimpedance amplifier.
EPL Photometer This is a free field photometer for measuring optical stimulation in startle setups. It's response range includes some UV.
EPL Equalizer This is a two channel filter intended for use in our EPL Cochlear Function Testing System
EPL Bipolar Current Source This current source is for driving cochlear implant electrodes. It has very low DC leakage, a monitor output, and is lab grade with no batteries required.
EPL Monopolar Current Source This is a single quadrant current source for driving LEDs in startle and other behavioral studies. It has 3 versions (20mA, 30mA, 300mA) and an optional current monitor output.
EPL Valve Driver This is exactly what it says. The valves are commonly used in behavioral studies.
EPL 2 Grass Adapter This is an adapter from our standard EPL power supply to the Grass Instruments 4 pin power connector common from the late 1970s to early 2010s.
EPL Cardiotach A heart rate monitor for veterinary use. It includes an LCD read out, integrated windowed warning alarms, and optional remote output with linear analog output to a data logger.
EPL Wearable Microfluidic Pump Hardware and Firmware The first wearable intracochlear drug delivery pump ever made for subjects of this size. This is the most recent poster to be co-authored with the engineering department. Please note a lot of this data was gathered with preliminary versions of the device. Poster
EPL/Ext. Lick-O-Meter This device electrically senses when a probe has been licked. This is really not our work, we just added stuff to integrate it into our subrack ecosystem.
EPL/TDT Headstage Breakout This is a breakout board for one version of TDT headstage. Warning: It worked for the investigator but not all the pin labels are guaranteed to be correct.
EPL Differential Amplifier This is a selectable gain, labratory grade (long term stable, precsion), amplifier that can drive a long (40ft) cable.
EPL Differential Buffer This is a unity gain differential amplifier that can drive a 40ft cable. It is commonly used with the EPL Bipolar Current Sources to monitor the stimuli delivered.
EPL Cross Point Switch This is a cross point switch used to multiplex multiple EPL Bipolar Current Sources across an even larger number of implant electrodes via computer control.
EPL Subrack and EPL Subrack Module This is the standard rack setup we developed on top of a commercially available subrack for use in the lab. Most of our hardware is in this ecosystem.
EPL Chamber heat This is not so much a project as documentation of our various struggles with the aging heating systems of our chambers and our attempts working on newer designs for additional chambers.


Note: Not listed are our retired projects which were ether in service for years often decades, occasionally used and quickly discarded, or canceled.
